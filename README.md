# Smart City Dashboard

## Overview
Smart City Dashboard is a collaborative full-stack web application developed as an internship project. It provides a centralized dashboard for managing and monitoring various smart city services.

## Features
- Admin Dashboard
- Citizen Dashboard
- User Authentication
- Responsive UI
- Modern React + Vite Frontend

## Tech Stack
- React
- Vite
- JavaScript
- HTML
- CSS

## Installation

```bash
npm install
npm run dev
```

## Team
Developed as part of an internship project.
